# WeatherNow ⛅

A React Native weather app built with Expo.

## Features

* Real-time weather with GPS detection
* 7-day weather forecast
* 24-hour temperature bar chart
* Dark / Light theme
* Temperature unit toggle (°C / °F)
* Recent search history

## Tech Stack

* React Native + Expo
* OpenWeatherMap API
* AsyncStorage
* Lottie Animations
* React Navigation

## Setup

1. Clone the repository


git clone https://github.com/S-mounika14/Weather-app.git


2. Install dependencies


npm install


3. Create `.env` file


EXPO_PUBLIC_OWM_KEY=your_openweathermap_api_key_here


4. Run the app


npx expo start
